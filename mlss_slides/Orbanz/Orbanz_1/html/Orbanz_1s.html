Probability: Definitions
Def: Measurable mapping
Let A, B be σ-algebras in Λ, Ω. A mapping X : Λ → Ω is called
measurable if X −1(B) ∈ A for all B ∈ B.
Interpretation: “F measurable” means that expression
“P(X −1(B))” makes sense.
Def: Random variables
A random variable X is a measurable mapping from an abstract
probability space (Λ, A, P) into an observation space (Ω, B(Ω)).
Image Measure
The measure P is not known explicitly. We work with the
distribution µX of random variable X defined as the image

Parametric Model
Parametric model
Let X : (Λ, A) → (Ωx , Bx ) and Θ : (Λ, A) → (Ωθ, Bθ) be two

I µΘ := Θ(P) the prior measure

The Problem with Kolmogorov
Problem
If dimension E is uncountable, the projective limit measure µE is
basically useless.
Explanation
I Domain of µE: BE (generated by product topology)
I Sets in BE: “axes-parallel” in all but countably many
dimensions
I E uncountable → BE too coarse for meaningful modeling
A Note of Caution:
Problem is often neglected in literature.
Example: Original paper on the DP (Ferguson, 1973).
23 / 25
Uncountable Dimensions
Intuition:
Objects of interest effectively have countably many degrees of
freedom.
Examples
I Continuous functions: Completely defined by values on
dense subset (e.g. Q in R)
I Probability measures: Completely defined by values on
countable system of sets.
Strategies
1. Modify theorem to directly define measure on “interesting”
space (eg space of continuous functions).
2. Use Kolmogorov theorem, than restrict µE to interesting
subspace.
